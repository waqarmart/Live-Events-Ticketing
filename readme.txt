== Installation ==

Thanks for downloading and installing "Live Events Ticketing" Plugin.

Read the documentation by going to this url below:
https://demo.skpsoft.com/documentation.html

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **Plugin Check**.
3. Install and activate the Plugin Check plugin.

= Manual installation =

1. Upload the entire `plugin-check` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Plugin Check plugin.
